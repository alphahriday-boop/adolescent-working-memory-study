# adolescent-working-memory-study
Adolescent Working Memory Study
Randomised, single-blind, within-subjects crossover experiment investigating whether a 250ml Red Bull energy drink produces greater working memory improvements than plain water in teenagers aged 13–17.
What I built
Two computerised cognitive assessments deployed via PsyToolKit:

2-back task (Red_Bull_vs__water.html) — participants viewed sequential letters and identified when the current letter matched one shown two trials prior. Measures spatial working memory and updating ability.
Word Span Backward — verbal working memory task where participants repeated spoken word sequences in reverse order.

Study design

8 participants, ages 13–17
Randomised counterbalanced order (Red Bull first vs. water first)
7-day washout period between sessions
30-minute absorption window post-consumption before testing
Parental consent and participant assent required for all participants
Single-blind: researcher blind to beverage assignment during testing

Results
No statistically significant difference between conditions on either measure:

WSB: t(7) = 1.369, p = .213, d = 0.48
2-back: t(7) = 0.391, p = .708, d = 0.14

A small-to-medium trend favoured Red Bull on the verbal task (d = 0.48), consistent with adult literature, but likely underpowered at n=8. Null results don't mean no effect — they mean the study needs replication with ~36 participants.
